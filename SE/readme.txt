### 物品“复活”软件 README

#### 介绍

物品“复活”软件是一个基于命令行的 Python 程序，帮助用户管理闲置物品信息，方便查找、添加、删除和显示物品列表。通过保存物品名称、描述和联系人信息，用户可以避免浪费物品，将它们分享给其他有需要的人。

#### 功能
- **添加物品**：将物品的名称、描述和联系人信息添加到物品列表中。
- **删除物品**：根据物品名称删除该物品的信息。
- **显示物品列表**：查看所有已存储的物品及其详细信息。
- **查找物品**：根据物品名称查找详细信息。
- **数据持久化**：物品数据保存到文件，下次启动时自动加载。

#### 使用方法

##### 1. 安装与运行
- 确保系统已安装 Python 3.x。
- 下载或克隆本项目到本地。
- 打开终端（Windows 上可使用 CMD 或 PowerShell），进入程序所在目录。
- 运行以下命令启动程序：
  ```bash
  python3 SE/hw_item_resurrection.py
  ```

##### 2. 操作流程

程序运行后，将显示操作菜单。根据菜单提示输入对应编号执行相应的操作：
1. **添加物品**：输入物品名称、描述和联系人信息。
2. **删除物品**：根据物品名称删除物品。
3. **显示物品列表**：展示所有存储的物品信息。
4. **查找物品**：根据物品名称查找对应物品的详情。
5. **退出程序**：程序退出并自动保存所有数据到文件中。

#### 文件说明

- **`SE/hw_item_resurrection.py`**: 主程序文件，包含所有功能实现。
- **`items_data.json`**: 物品数据存储文件，程序自动生成并保存用户添加的物品信息。
