物品管理

介绍

这是一个基于 Python 的物品管理系统，用于记录和管理物品信息。用户可以通过该系统进行物品的增、删、查、改等操作，同时程序会将每次操作记录到日志文件中，方便追溯和调试。系统的数据保存在 JSON 文件中，以确保数据的持久性。

功能

- 加载数据：启动时加载存储在 items_data.json 文件中的物品数据。
- 添加物品：可以添加物品的名称、描述和联系人信息。
- 删除物品：可以删除物品。
- 修改物品：可以修改已存在物品的描述或联系人信息。
- 查找物品：可以根据物品名称查找物品信息。
- 显示物品列表：显示所有当前物品的信息。
- 日志记录：所有操作会记录到 operation_log.txt 文件中，包括操作类型和相关详情。

环境要求

- Python 3.x
- 需要访问文件系统的权限，以便读写数据文件和日志文件。

文件结构

物品管理系统/
│
├── items_data.json        # 存储物品数据的 JSON 文件
├── operation_log.txt      # 操作日志文件
├── hw_item_resurrection.py.py                # 主程序代码

使用方法

1. 克隆或下载代码到本地。
2. 确保 Python 环境已安装。
3. 启动程序：
   python .../hw_item_resurrection.py

操作流程

程序启动后，用户会看到以下操作菜单：

请选择操作:
1. 添加物品
2. 删除物品
3. 显示物品列表
4. 查找物品
5. 修改物品信息
6. 退出

- 添加物品：输入物品名称、描述和联系人信息，系统将会把物品添加到数据中，并保存到文件。
- 删除物品：输入物品名称，系统将删除指定的物品。
- 显示物品列表：系统将列出所有物品的名称、描述和联系人信息。
- 查找物品：输入物品名称，系统将查找并显示物品信息。
- 修改物品信息：输入物品名称，选择修改描述或联系人信息。
- 退出：退出程序，并保存所有更改。

日志记录

所有操作都会记录到 operation_log.txt 文件中。每次操作记录会包含时间戳、操作类型和详情。例如：

2024-11-14 10:15:34,562 - INFO - 操作: 添加物品, 详情: 物品 '书包' 添加成功
2024-11-14 10:16:45,324 - INFO - 操作: 修改物品, 详情: 物品 '书包' 描述已修改为：红色运动书包
2024-11-14 10:17:56,431 - INFO - 操作: 删除物品, 详情: 物品 '书包' 已删除
